# Data-Science-projects
This repository contains two data science projects focused on database design and data analysis.

1) Restaurant Database (SQL): Designed and implemented databases for a restaurant system.
-Designed and created an OLTP database to handle order-related data.
-Developed an OLAP database with an ETL process for analytical queries and reporting.
-Created triggers to manage real-time updates.

2) Weather Data Analysis (Python): Explored and visualized weather data to identify trends and patterns in temperature across regions.
-Cleaned and transformed raw data using Pandas.
-Created visualizations with Matplotlib.
